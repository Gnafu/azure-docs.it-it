# <a name="box-and-azure-active-directory"></a>Box e Azure Active Directory

Accelera l'adozione di box nell'organizzazione e aumenta l'adozione da parte degli utenti grazie all'integrazione precompilata. Consente agli utenti di collaborare rapidamente tramite la connessione Box alla Azure Active Directory (Azure AD).

![Mostra il logo dell'app box](./media/box.png)

> [!div class="nextstepaction"]
> [Casella try con Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Che cosa è possibile fare con box e Azure AD?

- **Easy Single Sign-on** : Elimina più nomi utente e password e semplifica l'accesso consentendo agli utenti di accedere a box usando i propri account Azure ad.
- Automatizzare il provisioning **utenti** : automatizzare la creazione, la manutenzione e la rimozione degli account utente in box quando gli utenti si uniscono, spostano o lasciano l'organizzazione.
- **Proteggere, controllare e monitorare** l'accesso sicuro a Box applicando criteri di accesso condizionale, governare l'accesso tramite le verifiche di accesso e monitorare l'accesso con i log di accesso.

## <a name="get-started"></a>Attività iniziali

Configurare il [provisioning utenti](https://aka.ms/BoxSSODoc) e l' [accesso Single Sign-on](https://aka.ms/BoxProvisioningDoc) per box con le guide di configurazione semplici.
