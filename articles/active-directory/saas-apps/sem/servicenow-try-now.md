# <a name="servicenow-and-azure-active-directory"></a>ServiceNow e Azure Active Directory

Accelera l'adozione di ServiceNow nell'organizzazione e aumenta l'adozione dell'utente con l'integrazione predefinita. Semplifica la Gestione dei servizi IT connettendo ServiceNow a Azure Active Directory (Azure AD).

![Mostra il logo dell'app ServiceNow](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Prova ServiceNow con Azure AD >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Che cosa è possibile fare con ServiceNow e Azure AD?

- **Easy Single Sign-on** : Elimina più nomi utente e password e semplifica l'accesso consentendo agli utenti di accedere a ServiceNow usando i propri account Azure ad.
- **Automatizzare** il provisioning degli utenti: automatizzare la creazione, la manutenzione e la rimozione degli account utente in ServiceNow quando gli utenti si uniscono, spostano o lasciano l'organizzazione.
- **Proteggere, controllare e monitorare** l'accesso protetto a ServiceNow applicando criteri di accesso condizionale, governare l'accesso tramite le verifiche di accesso e monitorare l'accesso con i log di accesso.

## <a name="get-started"></a>Attività iniziali

Configurare il [provisioning utenti](https://aka.ms/ServiceNowProvisioning) e l' [accesso Single Sign-on](https://aka.ms/ServiceNowSSO) per ServiceNow con le guide di configurazione semplici.
