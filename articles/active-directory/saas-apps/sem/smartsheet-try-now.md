# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet e Azure Active Directory

Accelera l'adozione di Smartsheet nell'organizzazione e aumenta l'adozione dell'utente con l'integrazione predefinita. Consentire agli utenti di collaborare rapidamente con la connessione di Smartsheet a Azure Active Directory (Azure AD).

![Mostra il logo dell'app Smartsheet](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Provare Smartsheet con Azure AD >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Che cosa è possibile fare con Smartsheet e Azure AD?

- **Easy Single Sign-on** : Elimina più nomi utente e password e semplifica l'accesso consentendo agli utenti di accedere a Smartsheet usando i propri account Azure ad.
- **Automatizzare** il provisioning degli utenti: automatizzare la creazione, la manutenzione e la rimozione degli account utente in Smartsheet quando gli utenti si uniscono, spostano o lasciano l'organizzazione.
- **Proteggere, controllare e monitorare** l'accesso protetto a Smartsheet applicando criteri di accesso condizionale, governare l'accesso tramite le verifiche di accesso e monitorare l'accesso con i log di accesso.

## <a name="get-started"></a>Attività iniziali

Configurare il [provisioning utenti](https://aka.ms/SmartsheetProvisioningDoc) e l' [accesso Single Sign-on](https://aka.ms/SmartsheetSSODoc) per Smartsheet con le semplici guide alla configurazione.
