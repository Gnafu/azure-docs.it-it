# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>Visualizzare le attività di accesso recenti nel portale Profilo personale (anteprima)
È possibile visualizzare tutte le attività di accesso recenti usando il proprio account aziendale o dell'istituto di istruzione nella pagina **Attività recenti** del portale **Profilo personale**.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>Visualizzare le attività di accesso recenti
1. Accedere all'account aziendale o dell'istituto di istruzione e quindi passare alla pagina https://myprofile.microsoft.com/.

2. Selezionare **Attività recenti** nel riquadro di spostamento sinistro oppure selezionare il collegamento **Verifica l'attività** nel blocco **Attività recenti**.

    ![Pagina Profilo personale con il collegamento Attività recenti evidenziato](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. Espandere ed esaminare ogni attività di accesso, assicurandosi di riconoscerle tutte. Se si hanno dubbi su un'attività di accesso, è consigliabile cambiare la password per proteggere l'account, in caso sia stato compromesso.

    ![Pagina Attività recenti con i dettagli di accesso espansi](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >Se si ritiene che il proprio account sia stato compromesso, vedere [Come accedere a un account Microsoft compromesso](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account) per altre informazioni su come reimpostare la password o sbloccare l'account.

## <a name="next-steps"></a>Passaggi successivi

- Dopo aver visualizzato le attività di accesso recenti, è possibile scegliere di visualizzare o gestire le [informazioni di sicurezza](user-help-security-info-overview.md), i [dispositivi](myprofile-portal-devices-page.md) o le [organizzazioni](myprofile-portal-organizations-page.md) connesse oppure di controllare in che modo l'organizzazione usa i [dati dell'utente](myprofile-portal-privacy-page.md).
