# <a name="preview-upcoming-changes-to-azure-application-insights"></a>Visualizzare in anteprima le modifiche future in Azure Application Insights 

Per Application Insights vengono rilasciate di frequente nuove funzionalità. Per visualizzare le anteprime di questi miglioramenti, è possibile registrarsi nel pannello di anteprima di Application Insights.  Il team di sviluppo rende disponibili le anteprime delle nuove funzionalità su base limitata prima di rilasciarle a tutti gli utenti. 

La figura seguente illustra come impostare le preferenze di anteprima.

![Pannello di anteprima di Application Insights](./media/app-insights-preview/preview.png)

## <a name="set-preferences"></a>Impostare le preferenze

Nel pannello di anteprima è possibile selezionare una delle opzioni seguenti relative alla visualizzazione delle anteprime.

- Sempre: le funzionalità in anteprima sono visualizzate non appena disponibili.
- Automatico: sono visualizzate le funzionalità in anteprima consigliate da Microsoft per l'account. 
- Mai: sono visualizzate sole le funzionalità in anteprima selezionate.
