# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Come gestire il piano tariffario dell'account Mappe di Azure

È possibile gestire il piano tariffario dell'account Mappe di Azure tramite il portale di Azure. Dopo aver creato un [account](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) è possibile visualizzare o cambiare il piano tariffario dell'account.

Per altre informazioni sulla scelta del piano tariffario adeguato, vedere [Scelta del piano tariffario corretto](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-pricing-tier"></a>Visualizzare il piano tariffario

Per visualizzare il piano tariffario scelto, passare all'opzione **Piano tariffario** nel menu delle impostazioni.

![Visualizzare il piano tariffario scelto](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>Modificare il piano tariffario

Dopo la creazione dell'account Mappe di Azure è possibile effettuare l'aggiornamento o il downgrade del piano tariffario per l'account. Per effettuare l'aggiornamento o il downgrade, passare all'opzione **Piano tariffario** nel menu delle impostazioni e selezionare il piano tariffario che non è evidenziato. Fare clic sul pulsante Salva per salvare l'opzione di piano tariffario selezionata.

![Modificare il piano tariffario](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Non è necessario generare nuove chiavi di sottoscrizione quando si effettua l'aggiornamento o il downgrade del piano tariffario per l'account Mappe di Azure.
