# <a name="interact-with-the-map---mouse-events"></a>Interazione con la mappa - eventi del mouse

Questo articolo illustra come usare [addEventListener](https://docs.microsoft.com/javascript/api/azure-maps-control/atlas.map?view=azure-iot-typescript-latest#addeventlistener) per rendere interattiva la mappa.
