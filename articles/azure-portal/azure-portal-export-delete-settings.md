# <a name="export-or-delete-settings-in-the-azure-portal"></a>Esportare o eliminare le impostazioni nel portale di Azure
È possibile esportare ed eliminare le impostazioni seguenti dal menu delle impostazioni del portale di Azure:
* Dashboard privati del portale di Azure
* Impostazioni utente, ad esempio sottoscrizioni o directory preferite e ultimo accesso alla directory
* Temi e personalizzazioni

È consigliabile esportare ed esaminare le impostazioni prima di eliminarle. Ricompilare i dashboard e aggiungere nuovamente le personalizzazioni potrebbe richiedere tempo.

[!INCLUDE [GDPR-related guidance](../../includes/gdpr-intro-sentence.md)]

**Esportare o eliminare le impostazioni del portale**

1. Accedere al [portale di Azure](http://portal.azure.com).
2. In alto a destra nel portale selezionare **Impostazioni**.

    ![Screenshot che mostra l'ingranaggio per le impostazioni del portale](media/azure-portal-export-delete-settings/azure-portal-settings-icon.png)
3. Selezionare **Esporta tutte le impostazioni** o **Elimina tutte le impostazioni e i dashboard privati**.

    ![Screenshot che mostra l'esportazione e l'eliminazione delle impostazioni del portale](media/azure-portal-export-delete-settings/azure-portal-export-delete-settings.png)

      La tabella seguente descrive le singole impostazioni. 

      | Impostazione | DESCRIZIONE |
      | --- | --- |
      | **Esporta tutte le impostazioni** | Crea un file JSON contenente le impostazioni utente, ad esempio il tema del dashboard.|
      | **Elimina tutte le impostazioni e i dashboard privati** | Elimina tutti i collegamenti ai dashboard privati e alle altre personalizzazioni utente eseguite nel dashboard del portale. |
