# <a name="introduction-to-azure-stack-networking"></a>Introduzione alla rete di Azure Stack

*Si applica a: Azure Stack integrate di sistemi Azure Stack Development Kit*

Stack di Azure offre un'ampia gamma di funzionalità di rete che può essere usato insieme o separatamente:

- **Connettività tra le risorse di Azure Stack**  
    Connettere le risorse di Azure insieme in una rete virtuale privata protetta nel cloud.
- **Connettività Internet**  
    Comunicazione da e verso le risorse di Azure Stack tramite Internet.
- **Connettività locale**  
    Connettere una rete locale alle risorse di Azure Stack tramite una rete privata virtuale (VPN) tramite Internet o tramite una connessione dedicata allo Stack di Azure.
- **Bilanciamento del carico e direzione del traffico**  
    Caricare bilanciare il traffico ai server nello stesso percorso e il traffico diretto ai server in posizioni diverse.
- **Sicurezza**  
    Filtrare il traffico di rete tra subnet della rete o singole macchine virtuali (VM).
- **Routing**  
    Utilizzare il routing predefinito o controllare completamente il routing tra Stack di Azure e alle risorse locali.
- **Facilità di gestione**  
    Monitorare e gestire le risorse di rete dello Stack di Azure.
- **Strumenti di configurazione e distribuzione**  
    Usare un portale basato sul web o gli strumenti da riga di comando multipiattaforma per distribuire e configurare le risorse di rete.
