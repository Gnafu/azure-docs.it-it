# <a name="how-to-pay-your-bill-for-microsoft-azure"></a>Come pagare la fattura per Microsoft Azure

Questo articolo si applica ai clienti con un contratto di Microsoft dal cliente che ha effettuato l'iscrizione ad Azure tramite il sito Web di Azure. [Controllare l'accesso a un contratto di Microsoft dal cliente](#check-access-to-a-microsoft-customer-agreement).

Il metodo di pagamento predefinito del profilo di fatturazione può essere una carta di credito o controllo/transito trasferimento. Se si dispone di crediti di Azure, questi vengono applicati automaticamente per la tua fattura ogni periodo di fatturazione. Per diventare idoneo per il pagamento tramite controllo/bonifico bancario, vedere [come pagare tramite fattura](billing-how-to-pay-by-invoice.md).

Se il metodo di pagamento predefinito è check/bonifico, è possibile inviare il pagamento all'indirizzo elencato nella parte inferiore della fattura. In alternativa, è possibile pagare online con una carta di credito nel portale di Azure.

Se il metodo di pagamento predefinito è una carta di credito, saranno automaticamente addebitate sulla carta di credito ogni periodo di fatturazione dopo la generazione della fattura. È anche possibile acquistare online nel portale di Azure se si desidera usare una carta di credito diversa o se l'addebito automatica ha esito negativo.

## <a name="pay-now-in-the-azure-portal"></a>Pagamento immediato nel portale di Azure

È necessario essere un proprietario del profilo di fatturazione, collaboratore o gestore fattura per pagare le fatture nel portale di Azure.

1. Accedere al [portale di Azure](https://portal.azure.com).
1. Eseguire ricerche in **Gestione dei costi e fatturazione**.
1. Nel **Overview** pannello, fare clic sul *riepilogo* scheda.

    ![screenshot che Mostra scheda Riepilogo](./media/billing-how-to-pay-bill/billing-account-overview.png)

1. Se si dispone di una scadenza o scaduto invoice, si noterà l'importo totale dovuto.
1. Fare clic su blu **pagamento immediato** pulsante.

    ![screenshot che Mostra pulsante Paga adesso](./media/billing-how-to-pay-bill/paynow-button-overview.png)

1. Nel nuovo pannello, fare clic su **selezionare un metodo di pagamento** una carta di credito esistente o aggiungerne uno nuovo.<!-- screenshot of select payment method link -->
1. Dopo aver selezionato il metodo di pagamento, fare clic su **pagamento immediato**.

Lo stato della fattura verrà aggiornato a pagamento nel pannello delle fatture.
