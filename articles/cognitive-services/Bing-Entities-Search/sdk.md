# <a name="bing-search-sdk"></a>Bing Search SDK
Gli esempi dell'API Ricerca entità Bing includono scenari di:
1.  Ricerca di entità, ad esempio Tom Cruise, per ottenere informazioni dettagliate.
2.  Gestione della disambiguazione dei termini per le query con la possibilità di più finalità.
3.  Ricerca di un'entità locale, ad esempio un ristorante, per ottenere informazioni dettagliate su di essa.
4.  Ricerca di attività commerciali locali, ad esempio ristoranti, per ottenere informazioni dettagliate.
5.  Attivare la gestione di richieste non valide ed errori.

> [!NOTE] 
> Alcuni SDK sono ora in disponibilità generale, con modifiche alla documentazione in sospeso. 

I Bing Search SDK rendono facilmente accessibile la funzionalità di ricerca Web nei linguaggi di programmazione seguenti:
* Introduzione agli [esempi .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pacchetto NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.EntitySearch/1.2.0)
    * Vedere anche [Librerie .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingEntitySearch) per le definizioni e le dipendenze.
* Introduzione agli [esempi Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Vedere anche [Librerie Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/entitySearch) per le definizioni e le dipendenze.
* Introduzione agli [esempi Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Vedere anche [Librerie Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingentitysearch) per le definizioni e le dipendenze.
* Introduzione agli [esempi Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Vedere anche [Librerie Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-entitysearch) per le definizioni e le dipendenze.

Gli esempi dell'SDK per ogni linguaggio includono un file Leggimi con i dettagli sui prerequisiti e sull'installazione e l'esecuzione degli esempi.
