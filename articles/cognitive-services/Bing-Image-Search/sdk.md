# <a name="bing-search-sdk"></a>Bing Search SDK
Gli esempi dell'API Ricerca immagini Bing includono scenari di:
1.  Query di un'immagine semplice.
2.  Restrizione della ricerca a immagini con tipi e proporzioni specifici, con funzionalità di filtro delle immagini.
3.  Ricerca di immagini di tendenza in quel momento in vari social network.
4.  Query di informazioni dettagliate su un'immagine, ad esempio origine, testo rappresentativo e così via.

I Bing Search SDK rendono facilmente accessibile la funzionalità di ricerca Web nei linguaggi di programmazione seguenti:
* Introduzione agli [esempi .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Pacchetto NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.ImageSearch/1.2.0)
    * Vedere anche [Librerie .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingImageSearch) per le definizioni e le dipendenze.
* Introduzione agli [esempi Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Vedere anche [Librerie Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) per le definizioni e le dipendenze.
* Introduzione agli [esempi Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Vedere anche [Librerie Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingimagesearch) per le definizioni e le dipendenze.
* Introduzione agli [esempi Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Vedere anche [Librerie Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) per le definizioni e le dipendenze.

Gli esempi dell'SDK per ogni linguaggio includono un file Leggimi con i dettagli sui prerequisiti e sull'installazione e l'esecuzione degli esempi.
