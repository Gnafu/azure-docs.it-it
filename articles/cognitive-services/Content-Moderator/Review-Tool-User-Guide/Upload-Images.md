# <a name="upload-images"></a>Caricare immagini

Per esplorare l'interfaccia utente, caricare fino a cinque immagini alla volta. La dimensione del file di ogni immagine non deve superare 2 MB. Controllare la dimensioni dei file prima del caricamento.
