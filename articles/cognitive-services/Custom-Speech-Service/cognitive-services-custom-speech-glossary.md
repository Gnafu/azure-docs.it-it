# <a name="glossary"></a>Glossario

## <a name="a"></a>Una 

### <a name="acoustic-model"></a>Modello acustico

Il modello acustico è un classificatore che etichetta brevi frammenti di audio in una serie di fonemi, o unità di suono, in una lingua specifica. Ad esempio, la parola inglese "speech" è costituita da quattro fonemi "s p iy ch". Queste classificazioni vengono effettuate con una frequenza di 100 volte al secondo.

## <a name="c"></a>C

### <a name="conversational-model"></a>Modello colloquiale

Modello adatto per il riconoscimento vocale in una conversazione. Il modello acustico colloquiale Microsoft è adattato per il parlato solitamente diretto a un'altra persona.

## <a name="d"></a>D

### <a name="deployment"></a>Distribuzione

Processo tramite il quale il modello personalizzato adattato diventa un servizio ed espone un URI.

## <a name="i"></a>I

### <a name="inverse-text-normalization"></a>Normalizzazione del testo inversa

Il processo di conversione di testo non elaborato e non formattato in testo formattato, ad esempio con maiuscole e punteggiatura, è denominato normalizzazione del testo inversa.

## <a name="j"></a>J

## <a name="k"></a>K

## <a name="l"></a>L

### <a name="language-model"></a>Modello linguistico

Il modello linguistico è una distribuzione probabilistica di sequenze di parole. Il modello linguistico aiuta il sistema a decidere tra sequenze di parole con suoni simili, in base alla probabilità delle sequenze di parole stesse.

## <a name="m"></a>M

## <a name="n"></a>N

### <a name="normalization"></a>Normalizzazione

Normalizzazione (testo) : trasformazione del testo risultante (ossia trascrizione) in una forma standard, non ambigua leggibile dal sistema.

## <a name="s"></a>S

### <a name="search-and-dictate-model"></a>Modello di ricerca e dettatura

Modello acustico appropriato per l'elaborazione di comandi. Il modello acustico Microsoft per ricerca e dettatura è appropriato per un parlato diretto a un'applicazione o un dispositivo, ad esempio comandi.

### <a name="subscription-key"></a>Chiave della sottoscrizione

Stringa che deve essere specificata come parametro di stringa di query per richiamare un qualsiasi modello del Servizio di riconoscimento vocale personalizzato. Una chiave di sottoscrizione si ottiene dal [portale di Azure](https://portal.azure.com/#create/Microsoft.CognitiveServices/apitype/CustomSpeech) e una volta ottenuta è disponibile in Sottoscrizioni personali nel portale del Servizio di riconoscimento vocale personalizzato.

## <a name="t"></a>T

### <a name="transcription"></a>Trascrizione

Frammento di testo risultante dall'elaborazione di un frammento di un file WAV audio.
