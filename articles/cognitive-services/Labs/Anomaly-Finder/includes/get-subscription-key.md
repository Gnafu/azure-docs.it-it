Prima di creare l'esempio, è necessario sottoscrivere l'[API Ricerca di anomalie](https://labs.cognitive.microsoft.com/en-us/project-anomaly-finder), che fa parte di Servizi cognitivi Microsoft. Fare clic sul pulsante "Subscribe" (Sottoscrivi), quindi accedere con l'account Microsoft (Hotmail, ecc.) per sottoscrivere l'API. Le chiavi di abbonamento gratuite saranno disponibili [qui](https://labs.cognitive.microsoft.com/en-us/subscriptions?productId=/products/AnomalyDetection.Preview&source=labs). È possibile usare indifferentemente la chiave Key 1 o Key 2.
