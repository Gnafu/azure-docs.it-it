> [!NOTE]
> Ottenere prima una chiave di sottoscrizione.
> È possibile ottenere le chiavi di sottoscrizione della versione di valutazione gratuita dalla pagina di [sottoscrizione di Servizi cognitivi](https://azure.microsoft.com/try/cognitive-services/). Dopo avere selezionato Speech API scegliere **Ottenere una chiave API** per ottenere due chiavi (primaria e secondaria). È possibile usare una delle due chiavi. Entrambe le chiavi sono associate alla stessa quota.
>
> Le chiavi di sottoscrizione a pagamento sono disponibili tramite il dashboard di Azure.
