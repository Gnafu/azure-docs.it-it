# <a name="what-is-cognitive-services"></a>Che cosa sono i servizi cognitivi?

I Servizi cognitivi Microsoft (in precedenza Project Oxford) sono costituiti da un set di API, SDK e servizi che aiutano gli sviluppatori a rendere le applicazioni più intelligenti, coinvolgenti e individuabili. Servizi cognitivi Microsoft si basa sul portafoglio in evoluzione di Microsoft di API di Machine Learning e consente agli sviluppatori di aggiungere facilmente funzionalità intelligenti, ad esempio il rilevamento di emozioni e video, il riconoscimento facciale, vocale e visivo, nelle applicazioni. Gli obiettivi sono un'esperienza di elaborazione più personale e una maggiore produttività supportate da sistemi che sempre più riescono a vedere, sentire, parlare, comprendere e persino iniziare a ragionare.

La documentazione di ogni prodotto la documentazione di riferimento dell'API corrispondente si trovano nella tabella di navigazione a sinistra.

## <a name="getting-started-with-free-trials"></a>Introduzione alle versioni di valutazione gratuite
Per iscriversi alle versioni di valutazione gratuite sono sufficienti un messaggio di posta elettronica e alcuni[semplici passaggi](https://azure.microsoft.com/try/cognitive-services/ "Guida all'iscrizione"). Sarà necessario un account Microsoft se non ne esiste già uno. Si riceverà una coppia di chiavi univoca per ogni API richiesta. La seconda è semplicemente di riserva. Non condividere le chiavi private con nessuno. Le versioni di valutazione hanno un limite di frequenza, in termini di transazioni al secondo o al minuto, e un limite di utilizzo mensile. Una transazione è semplicemente una chiamata API. È possibile eseguire l'aggiornamento ai livelli a pagamento per sbloccare le restrizioni.

## <a name="subscription-management"></a>Gestione sottoscrizioni
Dopo l'accesso con l'account Microsoft sarà possibile accedere a [Sottoscrizioni personali] (https://www.microsoft.com/cognitive-services/en-us/subscriptions "Sottoscrizioni personali") per visualizzare i prodotti usati, la quota rimanente e la possibilità di aggiungere altri prodotti alla sottoscrizione.

## <a name="upgrade-to-unlock-limits"></a>Eseguire l'aggiornamento per sbloccare i limiti
Tutte le API avranno un piano di valutazione gratuita.  Non appena le offerte a pagamento saranno disponibili per ogni API, si verrà reindirizzati al portale di Azure per completare l'acquisto.  È possibile trovare i collegamenti *Acquista* nella pagina Sottoscrizioni se le si sta già usando oppure è possibile ignorare le versioni di valutazione e procedere all'acquisto con i collegamenti disponibili in [Prezzi](https://www.microsoft.com/cognitive-services/en-us/pricing "Prezzi").  Sarà necessario configurare un account sottoscrittore di Azure con una carta di credito e un numero di telefono. Se si hanno particolari requisiti o si vuole semplicemente comunicare con gli addetti alle vendite, fare clic sul pulsante "Contattaci" nella parte superiore della pagina dei prezzi.

## <a name="regional-availability"></a>Disponibilità internazionale
Le API nei Servizi cognitivi sono ospitate in una rete in espansione di centri dati gestiti da Microsoft. È possibile trovare la disponibilità a livello di area per ogni API nell'[elenco delle aree di Azure](https://azure.microsoft.com/regions).
    
Se si sta cercando un'area non ancora supportata, inviare una richiesta di funzionalità nel [forum di UserVoice](https://cognitive.uservoice.com/).

## <a name="support"></a>Supporto
Il supporto è disponibile durante tutto il processo di sviluppo. È possibile contattarci in qualsiasi momento usando i collegamenti seguenti. 
* Per assistenza e domande tecniche, pubblicare un post in [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-cognitive)
* Per commenti e suggerimenti e richieste di funzionalità, accedere a [UserVoice](https://cognitive.uservoice.com/)
* Per tutto il resto, fare clic su "Contattaci" nella parte inferiore di qualsiasi pagina
