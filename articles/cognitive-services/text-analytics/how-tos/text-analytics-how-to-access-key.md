# <a name="how-to-find-endpoints-and-access-keys-for-the-text-analytics-cognitive-service"></a>Come trovare gli endpoint e le chiavi di accesso per il servizio cognitivo Analisi del testo

Quando si effettua la registrazione per Analisi del testo, viene inviata una chiave di accesso personalizzata univoca per la sottoscrizione. Questa chiave è necessaria per ogni chiamata all'API Analisi del testo. Se la [registrazione](text-analytics-how-to-signup.md) non è stata ancora effettuata, eseguire l'operazione ora per ottenere la chiave. 

Per richiedere assistenza durante la ricerca della chiave o per determinare se la sottoscrizione include già Analisi del testo, usare le istruzioni seguenti per ottenere le informazioni necessarie. 

## <a name="find-your-service-endpoint-and-access-key"></a>Trovare l'endpoint di servizio e la chiave di accesso

1. Accedere al [portale di Azure](https://portal.azure.com).

2. Nel riquadro di spostamento a sinistra selezionare **Tutti i servizi**.

3. In Filtro digitare *Servizi cognitivi*. Se la sottoscrizione include Analisi del testo, viene visualizzata nell'elenco con **API Analisi del testo** come tipo di API.

4. Fare clic sul collegamento per aprire il pannello del servizio. È ora possibile ottenere una chiave da **Risorse > Chiavi** oppure facendo clic su **Mostra chiavi di accesso** nel riquadro Informazioni di base. Nel riquadro Informazioni di base è visibile anche l'endpoint.
