<tr><td>**Descrizione**</td><td>La libreria BulkExecutor consente alle applicazioni client di eseguire operazioni in blocco negli account di Azure Cosmos DB. La libreria BulkExecutor fornisce gli spazi dei nomi BulkImport e BulkUpdate. Il modulo BulkImport può inserire in blocco i documenti in maniera ottimizzata, in modo che la velocità effettiva di cui è stato eseguito il provisioning per una raccolta venga utilizzata nella misura massima consentita. Il modulo BulkUpdate può aggiornare in blocco i dati esistenti nei contenitori di Azure Cosmos DB come patch.</td></tr>
