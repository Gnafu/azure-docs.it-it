I white paper consentono di esplorare i concetti di Azure Cosmos DB a un livello più profondo. Questo articolo fornisce un elenco di white paper disponibili per Azure Cosmos DB.

| **Whitepaper** | **Descrizione** |
| --- | --- |
| [Costo totale di proprietà di un servizio Cloud di Database NoSQL](https://aka.ms/cosmosdb-tco-paper) |Questo white paper confronta il costo totale della proprietà e del funzionamento del software in diverse configurazioni di distribuzione, ad esempio un database on-premise, un database che usa l'infrastruttura cloud. Questo documento fornisce anche delle stime del costo totale di proprietà per i tipi di uso basso, medio e alto |
|[Eseguire l'indicizzazione dello schema-agnostico con Azure Cosmos DB](http://www.vldb.org/pvldb/vol8/p1668-shukla.pdf) | Questo documento descrive il sottosistema di indicizzazione di Azure Cosmos DB. Questo documento comprende le funzionalità di Azure Cosmos DB, ad esempio la rappresentazione del documento, il linguaggio della query, l'approccio di indicizzazione dei documenti, il supporto degli indici dei componenti di base e le esperienze di produzione anticipata.|
| [Azure Cosmos DB e dati personali](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers)| Questo documento fornisce indicazioni per i clienti di Azure Cosmos DB che gestiscono un database basato su cloud, un database on-premise, o entrambi e che hanno bisogno di verificare che i dati personali nei loro sistemi di database siano gestiti e protetti in base alle regole correnti. |
