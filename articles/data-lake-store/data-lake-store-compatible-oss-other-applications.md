# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-storage-gen1"></a>Applicazioni Big Data open source compatibili con Data Lake Storage Gen1

[!INCLUDE [data-lake-storage-gen1-rename-note.md](../../includes/data-lake-storage-gen1-rename-note.md)]

Questo articolo elenca le applicazioni Big Data di origine che funzionano con Azure Data Lake Storage Gen1. Per le applicazioni nella tabella seguente, sono supportate solo le versioni disponibili con la distribuzione elencata. Per sapere quali versioni di queste applicazioni sono disponibili con HDInsight, vedere [Controllo delle versioni del componente HDInsight](../hdinsight/hdinsight-component-versioning.md).

| Software Open Source | Distribuzione |
| --- | --- |
| [Apache Sqoop](http://sqoop.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [MapReduce](http://hadoop.apache.org/docs/r1.0.4/mapred_tutorial.html) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Storm](https://storm.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Hive](http://hive.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [HCatalog](https://cwiki.apache.org/confluence/display/Hive/HCatalog) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Mahout](http://mahout.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Pig/Pig Latin](http://pig.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Oozie](http://oozie.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Zookeeper](http://zookeeper.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Tez](http://tez.apache.org/) |HDInsight 3.2, 3.4, 3.5 e 3.6 |
| [Apache Spark](http://spark.apache.org/) |HDInsight 3.4, 3.5 e 3.6 |
