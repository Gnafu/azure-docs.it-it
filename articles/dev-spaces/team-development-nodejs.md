### <a name="make-a-code-change"></a>Effettuare una modifica del codice
Passare alla finestra di VS Code per `mywebapi` e apportare una modifica al codice per il gestore GET `/` predefinito, ad esempio:

```javascript
app.get('/', function (req, res) {
    res.send('mywebapi now says something new');
});
```
