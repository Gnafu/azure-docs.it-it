# <a name="register-your-azure-digital-twins-app-with-azure-active-directory-legacy"></a>Registrare l'app di dispositivi gemelli digitale di Azure con Azure Active Directory legacy

Questo articolo viene illustrato come che è possibile usare il modo obsolete o legacy per registrare l'applicazione di esempio per Azure Active Directory (Azure AD) in modo che possa accedere l'istanza di gemelli digitale. È possibile provare questo metodo in caso di Azure nuova registrazione di app AD non funziona per l'installazione.

[!INCLUDE [Digital Twins legacy AAD](../../includes/digital-twins-permissions-legacy.md)]

## <a name="next-steps"></a>Passaggi successivi

Dopo che l'app viene registrata con Azure ad, quindi possibile connettersi con l'istanza di gemelli digitali e della Guida che avanza nel proprio scenario. Entrambi i casi il [quickstart](quickstart-view-occupancy-dotnet.md#build-application) o il [esercitazione](tutorial-facilities-setup.md#configure-the-digital-twins-sample) per altre informazioni sui passaggi successivi.
