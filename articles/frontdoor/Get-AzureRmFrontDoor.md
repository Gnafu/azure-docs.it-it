# Get-AzureRmFrontDoor
Ottenere la configurazione per Frontdoor

## SINTASSI

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
**Get-AzureRmFrontDoor** cmdletGet recupera tutti i Frontdoor esistenti nella sottoscrizione corrente che corrispondano alle informazioni fornite.
