# New-AzureRmFrontDoorFireWallPolicy
Creare criteri di web application firewall

## SINTASSI

```
New-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> -Name <String> [-EnabledState <PSEnabledState>]
 [-Mode <PSMode>] -Customrule <PSCustomRule[]> [-ManagedRule <PSManagedRule[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzureRmFrontDoorFireWallPolicy** crea un nuovo criterio di web application firewall di Azure nel gruppo di risorse specificato della sottoscrizione corrente

## ESEMPI

### Esempio 1: Creare criteri di web application firewall
