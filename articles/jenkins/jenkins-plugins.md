# <a name="jenkins-plugins-for-azure"></a>Plug-in Jenkins per Azure

Il plug-in Jenkins seguenti supportano diverse funzionalità per l'uso con Azure.
  
| Plug-in Jenkins | DESCRIZIONE                                   |
|------------------------------------------------------------------------------|
| [Plug-in servizio App di Azure](https://plugins.jenkins.io/azure-app-service)     | Un plug-in di Jenkins per distribuire un servizio App di Azure (attualmente supporta solo l'App Web). | 
| [Plug-in Azure AD](https://plugins.jenkins.io/azure-ad)                       | Jenkins Plugin che supporta l'autenticazione e autorizzazione tramite Azure Active Directory. | 
| [Plug-in degli Agenti del servizio contenitore di Azure](https://plugins.jenkins.io/azure-container-agents) | Plug-in agenti di contenitore di Azure può essere utile per eseguire un contenitore come agente in Jenkins | 
| [Plug-in del servizio Azure Container](https://plugins.jenkins.io/azure-acs)       | Un Jenkins Plugin per distribuire le configurazioni di servizio contenitore di Azure (AKS). | 
| [Plug-in di Credenziali di Azure](https://plugins.jenkins.io/azure-credentials)      | Plug-in Jenkins per gestire le credenziali di Azure. | 
| [Plug-in di Funzioni di Azure](https://plugins.jenkins.io/azure-function)           | Per usare questo plug-in per distribuire funzioni di Azure, è necessario disporre di un'entità servizio di Azure nell'istanza di Jenkins. | 
| [Plug-in Service Fabric di Azure](https://plugins.jenkins.io/service-fabric)     | Un Jenkins Plugin per i progetti Linux di Azure Service Fabric. |
| [Plug-in di Archiviazione di Azure](https://plugins.jenkins.io/windows-azure-storage)     | Gli artefatti di compilazione di un plug-in per il caricamento o download di creare dipendenze da un archivio Blob di Microsoft Azure. | 
| [Plug-in agenti della macchina virtuale di Azure](https://plugins.jenkins.io/azure-vm-agents)         | Un Jenkins Plugin per creare agenti di Jenkins in macchine virtuali di Azure (tramite il modello di Azure Resource Manager). | 
| [Plug-in del set di scalabilità di macchine virtuali di Azure](https://plugins.jenkins.io/azure-vmss)           | Imposta un plug-in di Jenkins per distribuire le immagini di macchina virtuale per la scalabilità di macchine virtuali di Azure. |
