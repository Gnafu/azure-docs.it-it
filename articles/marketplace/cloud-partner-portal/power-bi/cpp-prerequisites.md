# <a name="power-bi-apps-prerequisites"></a>Prerequisiti delle app Power BI

Questo articolo elenca i requisiti tecnici e commerciali che è necessario soddisfare prima di poter pubblicare un'offerta di app Power BI in Microsoft AppSource Marketplace.

## <a name="technical-requirements"></a>Requisiti tecnici

Il principale asset tecnico che è necessario avere per questo tipo di offerta è un'[applicazione Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Per altre informazioni, vedere [Creare asset tecnici per un'offerta di app Power BI](./cpp-create-technical-assets.md).

Dopo aver creato e testato l'applicazione di servizio in Power BI, è necessario salvare l'URL di installazione dell'applicazione generato da Power BI. Questo URL è necessario per [creare una nuova offerta di app Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Requisiti aziendali

I requisiti commerciali includono obblighi procedurali, legali e contrattuali:

-   È necessario essere un editore di Marketplace cloud registrato. Se non si è ancora effettuata la registrazione, seguire la procedura nell'articolo [Diventare un editore di Marketplace cloud](../../become-publisher.md).

    > [!NOTE] 
    > Per l'accesso al [portale Cloud Partner](https://cloudpartner.azure.com) è consigliabile usare lo stesso account usato per la registrazione a Microsoft Developer Center. È consigliabile usare un solo account Microsoft per le offerte in AppSource. L'account non deve essere specifico per singoli servizi o offerte.
 
-   L'utente è responsabile di rendere il supporto tecnico disponibile per i clienti in una modalità ragionevole dal punto di vista commerciale.
-   I contenuti offerti devono soddisfare i criteri per la pubblicazione di offerte in AppSource. Per altre informazioni, vedere [Have an app to list on AppSource? Here's how](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how) (Come pubblicare un'app in AppSource).
-   È necessario rispettare l'[Informativa sulla privacy di Microsoft](https://privacy.microsoft.com/privacystatement).


## <a name="next-steps"></a>Passaggi successivi

Dopo aver soddisfatto tutti i requisiti, si è pronti per [creare un'offerta di Power BI](./cpp-create-offer.md) nel portale Cloud Partner.
