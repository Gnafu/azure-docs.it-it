# <a name="saas-application-channel-info-tab"></a>Scheda Informazioni canale Applicazione SaaS

Usare la scheda Informazioni canale per fornire informazioni sul canale per l'app, inclusi materiale GTM (Go-to-market) e le informazioni di contatto.

![Modulo Informazioni canale](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Per configurare le informazioni del canale

1. In **Informazioni canale> Materiali GTM**, immettere l'URL per il sito Web che ospita i materiali dai servizi GTM.
2. In **Channel Manager Contact** (Contatto gestore canali), fornire un contatto di gestione che Microsoft possa contattare per problemi di supporto tecnico e aziendale. Immettere le informazioni per i seguenti campi: Nome, Indirizzo e-mail e numero di telefono.
3. Selezionare **Salva** per completare la configurazione delle informazioni sul canale.
