# <a name="azure-saas-application-offer"></a>Offerta di applicazioni SaaS di Azure

Il [portale Cloud Partner](https://cloudpartner.azure.com/) non supporta più la creazione e la gestione delle offerte di applicazioni SaaS.  Al contrario, questa funzionalità sono state spostate da Microsoft [portale Centro per i Partner](https://partner.microsoft.com/pcv/).  Per altre informazioni, vedere [creare una nuova offerta SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Passaggi successivi

Prima di creare una nuova offerta di app SaaS, è consigliabile leggere il [applicazioni SaaS Guida alla pubblicazione dell'offerta](../../marketplace-saas-applications-technical-publishing-guide.md) ed esaminare i passaggi del portale Centro per i Partner elencati nella [elenco di controllo per la creazione offerta](../../partner-center-portal/offer-creation-checklist.md).
