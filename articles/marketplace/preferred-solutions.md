# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Soluzioni preferenziali in Azure Marketplace e AppSource

Una soluzione preferenziale Microsoft è un'applicazione cloud selezionata per la qualità, le prestazioni e la capacità di soddisfare le esigenze dei clienti in un determinato settore verticale o un'area della soluzione. Un team di esperti del settore e delle vendite Microsoft seleziona le soluzioni presentate da partner con [competenze Gold](https://partner.microsoft.com/membership/competencies) in Microsoft Partner Network. Tali soluzioni vengono offerte nelle vetrine marketplace cloud di Microsoft: Azure Marketplace e AppSource. Le soluzioni preferenziali in AppSource possono essere individuate per settori verticali, mentre in Azure Marketplace è possibile fare clic all'interno delle categorie dell'area della soluzione.

Microsoft contatterà i provider di soluzioni se una o più soluzioni sono state selezionate come preferenziali.
