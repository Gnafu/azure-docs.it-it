# <a name="common-power-bi-workspace-collection-scenarios"></a>Scenari comuni delle raccolte di aree di lavoro di Power BI

Le raccolte di aree di lavoro di Power BI hanno l'obiettivo di esporre la maggior parte delle funzionalità esistenti dell'API Power BI nell'ambito di un servizio di Azure utilizzabile per lo sviluppo delle applicazioni.  È anche possibile eseguire a livello di codice il provisioning, lo sviluppo e la distribuzione del contenuto di Power BI e delle risorse necessari. Per altre informazioni, vedere [Introduzione alle raccolte di aree di lavoro di Microsoft Power BI](get-started.md).

> [!IMPORTANT]
> Le raccolte di aree di lavoro di Power BI sono deprecate e sono disponibili fino a giugno 2018 o fino alla data specificata nel contratto. È consigliabile pianificare la migrazione a Power BI Embedded per evitare interruzioni nell'applicazione. Per informazioni su come eseguire la migrazione dei dati a Power BI Embedded, vedere [Come eseguire la migrazione del contenuto delle raccolte di aree di lavoro di Power BI a Power BI Embedded](https://powerbi.microsoft.com/documentation/powerbi-developer-migrate-from-powerbi-embedded/).

Di seguito sono riportati alcuni scenari per l'uso delle **raccolte di aree di lavoro di Power BI** nelle applicazioni.

* I report completamente interattivi che sono stati creati in Power BI Desktop possono essere incorporati nella propria applicazione usando un nuovo modello di autenticazione basato su token.
* I fornitori di software indipendenti e i clienti che compilano applicazioni destinate ai clienti possono usare il servizio **Raccolta di aree di lavoro di Power BI** e Power BI SDK per incorporare report interattivi. Per informazioni, vedere [Introduzione alle raccolte di aree di lavoro di Microsoft Power BI](get-started.md).
* Gli sviluppatori possono usare framework di visualizzazione di Power BI per creare visualizzazioni personalizzate che possono integrare nell'app. Vedere [Estendi Power BI con oggetti visivi personalizzati](https://powerbi.microsoft.com/custom-visuals/).
