# <a name="azure-network-security"></a>Sicurezza di rete di Azure
## <a name="abstract"></a>Sunto
I servizi di rete di Azure sono stati appositamente studiati per ottimizzare la flessibilità, la disponibilità, la resilienza, la sicurezza e l'integrità. Questo white paper presenta informazioni dettagliate sulle funzioni di rete di Azure. Descrive inoltre come i clienti possono usare le funzioni di sicurezza native in Azure per proteggere gli asset di informazioni.
I destinatari di questo white paper includono:
- Responsabili tecnici, amministratori di rete e sviluppatori che cercano soluzioni di sicurezza disponibili e supportate in Azure.
- PMI o responsabili dei processi aziendali che cercano una panoramica generale dei servizi e delle tecnologie di Azure correlati alla sicurezza di rete nel cloud pubblico di Azure.
