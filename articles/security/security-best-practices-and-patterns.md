# <a name="azure-security-best-practices-and-patterns"></a>Procedure consigliate e modelli per la sicurezza di Azure

L'articolo seguente contiene informazioni sulle procedure consigliate per la sicurezza da usare nella progettazione, distribuzione e gestione di soluzioni cloud tramite Azure. Le procedure consigliate si basano sulla nostra esperienza in materia di sicurezza Azure e dalle esperienze di altri clienti.

Le procedure consigliate costituiscono una risorsa per i professionisti IT. Potrebbe trattarsi di designer, architetti, sviluppatori e tester che creano e distribuiscono soluzioni sicure per Azure.

* [Procedure consigliate per la sicurezza della rete di Azure](azure-security-network-security-best-practices.md)
* [Procedure consigliate per la sicurezza operativa di Azure](azure-operational-security-best-practices.md)
* [Procedure consigliate per le distribuzioni PaaS di Azure](security-paas-deployments.md)
* [Procedure consigliate per la sicurezza di Azure Service Fabric](azure-service-fabric-security-best-practices.md)
* [Procedure consigliate per la sicurezza delle VM di Azure](azure-security-best-practices-vms.md)
* [Implementazione di un'architettura di rete ibrida sicura in Azure](../guidance/guidance-iaas-ra-secure-vnet-hybrid.md)
