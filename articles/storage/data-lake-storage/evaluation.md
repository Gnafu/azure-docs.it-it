# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Linee guida per i clienti di Azure Data Lake Storage Gen1

La versione di anteprima di Azure Data Lake Storage Gen2 offre funzionalità di archiviazione BLOB native, tra cui suddivisione in livelli dei dati, gestione del ciclo di vita e sicurezza di rete avanzata, oltre al file system compatibile con Hadoop disponibile in Data Lake Storage Gen1.

Microsoft consiglia ai clienti esistenti di Data Lake Storage Gen1 di valutare tutte le funzionalità nuove e migliorate di Data Lake Storage Gen2 e fornire commenti e suggerimenti [partecipando all'anteprima](https://aka.ms/adlsgen2signup). Continuerà a essere supportato anche Data Lake Storage Gen1. Data Lake Storage Gen1 e Gen2 possono coesistere, pertanto le risorse e le applicazioni esistenti che sfruttano Data Lake Storage Gen1 continuano a funzionare senza aggiornamenti. È possibile elaborare un piano per la migrazione a Data Lake Storage Gen2 a seconda delle proprie esigenze.

Se si sta valutando Data Lake Storage Gen1, Microsoft consiglia di partecipare all'anteprima di Gen2 per valutare le funzionalità nuove e migliorate offerte da Data Lake Storage Gen2. In modo analogo, i clienti di archivi BLOB esistenti sono invitati a valutare se un file system compatibile con Hadoop è adatto alle proprie esigenze di analisi.
