# <a name="activity-management-overview"></a>Panoramica dell'attività di gestione

CloudSimple tiene traccia di tutte le attività che possono influire sul funzionamento dei cloud privati. Queste attività, inclusi avvisi, eventi, le attività e attività di controllo. Il [pagine delle attività](https://docs.azure.cloudsimple.com/activity/) riepilogare tutte le attività correnti e consentono di eseguire il drill-per altri dettagli.

## <a name="events"></a>Eventi

Gli eventi di traccia utente e attività del sistema nel portale di CloudSimple.  Gli eventi vengono visualizzati l'attività associata a una risorsa specifica e la gravità dell'impatto.  È possibile visualizzare gli eventi dal portale CloudSimple.

## <a name="alerts"></a>Avvisi

Gli avvisi sono notifiche di qualsiasi attività nell'ambiente CloudSimple significativo.  Gli eventi, che influire sull'accesso utente o di fatturazione, vengono visualizzati come avvisi.  È possibile riconoscere gli avvisi generati dal portale CloudSimple.

## <a name="tasks"></a>Attività

Attività di tenere traccia di qualsiasi operazione utente, che richiede più di 30 secondi per il completamento.  È possibile monitorare lo stato di avanzamento di un'attività dal portale CloudSimple.  Completare Mostra le attività il tempo totale impiegato per il completamento dell'attività.

## <a name="audit"></a>Audit

L'utente ha eseguito le operazioni di traccia dei log di controllo.  I log contengono i parametri immessi dall'utente per l'operazione.  È possibile usare i log di controllo per monitorare l'attività di utente per tutti gli utenti.
