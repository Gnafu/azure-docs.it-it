### <a name="azure-notebooks---free-jupyter-based-notebooks-in-the-azure-cloud"></a>**Azure Notebooks**: notebook basati su Jupyter gratuiti nel cloud di Azure
L'SDK è già installato e configurato automaticamente in Azure Notebooks.
  
1. Completare la [guida introduttiva](../articles/machine-learning/service/quickstart-get-started.md) per creare un'area di lavoro e avviare Azure Notebooks.
1. Passare ad [Azure Notebooks](https://notebooks.azure.com/)
1. Nella libreria `Getting Started` creata durante la guida introduttiva passare alla cartella `tutorials`.
1. Aprire il notebook.

### <a name="your-own-jupyter-notebook-server"></a>**Server Jupyter Notebook personale**
1. Completare la [guida introduttiva su Python SDK](../articles/machine-learning/service/quickstart-create-workspace-with-python.md) per installare l'SDK e creare un'area di lavoro.
1. Clonare il [repository GitHub](https://aka.ms/aml-notebooks).
1. Copiare nella directory clonata la directory `aml_config` creata durante la guida introduttiva.
1. Avviare il server notebook dalla directory clonata.
1. Passare alla cartella `tutorials`.
1. Aprire il notebook.
