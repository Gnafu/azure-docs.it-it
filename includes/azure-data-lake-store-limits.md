Azure Data Lake Storage Gen1 è un repository su vasta scala a livello aziendale per carichi di lavoro di analisi di Big Data. Data Lake Storage Gen1 consente di acquisire dati di qualsiasi dimensione, tipo e velocità di inserimento in un'unica posizione per le analisi esplorative e operative. Non esiste alcun limite alla quantità di dati archiviabili in un account Data Lake Storage Gen1.

| **Risorsa** | **Limite predefinito** | **Commenti** |
| --- | --- | --- |
| Numero massimo di account Data Lake Storage Gen1 per sottoscrizione per ogni area |10 | Contattare il supporto per richiedere un aumento del limite |
| Numero massimo di ACL di accesso, per file o cartella |32 | Si tratta di un limite rigido. Per gestire l'accesso con un numero inferiore di voci, usare gruppi |
| Numero massimo di ACL predefiniti, per file o cartella |32 | Si tratta di un limite rigido. Per gestire l'accesso con un numero inferiore di voci, usare gruppi |
