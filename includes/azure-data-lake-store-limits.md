Azure Data Lake Storage Gen1 è un repository su vasta scala a livello aziendale per carichi di lavoro di analisi di Big Data. È possibile usare Data Lake archiviazione Gen1 per acquisire dati di qualsiasi dimensione, tipo e velocità di inserimento in un'unica posizione per analitica esplorative e operative. Non sussiste alcun limite alla quantità di dati che è possibile archiviare in un account Data Lake archiviazione Gen1.

| **Risorsa** | **Limite predefinito** | **Commenti** |
| --- | --- | --- |
| Numero massimo di account Data Lake archiviazione Gen1, per ogni sottoscrizione per area |10 | Per richiedere un aumento del limite, contattare il supporto tecnico. |
| Numero massimo di ACL di accesso, per ogni file o cartella |32 | Si tratta di un limite rigido. Usare gruppi per gestire l'accesso con un minor numero di voci. |
| Numero massimo di ACL predefiniti, per ogni file o cartella |32 | Si tratta di un limite rigido. Usare gruppi per gestire l'accesso con un minor numero di voci. |
