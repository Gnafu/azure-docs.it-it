> [!NOTE]
> Per ottenere risultati ottimali, quando si usa l'API rilevatore di anomalie, è necessario includere dati delle serie temporali in formato JSON:
> * punti dati separati dallo stesso intervallo, con non più del 10% del numero previsto di punti in cui Manca.
> * punti dati almeno 12 se i dati non hanno un criterio chiaro stagionale.
> * almeno 4 occorrenze pattern se i dati dispongono di un modello stagionale chiaro.
