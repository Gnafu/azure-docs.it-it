È necessario disporre di un [account API servizi cognitivi](../articles/cognitive-services/cognitive-services-apis-create-account.md) con accesso per l'API rilevatore di anomalie. Se non si ha una sottoscrizione di Azure, è possibile [creare un account](https://azure.microsoft.com/try/cognitive-services/) gratuito. È possibile ottenere la chiave di sottoscrizione dal [portale di Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md#access-your-resource) dopo aver creato l'account, o [sito Web di Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) dopo l'attivazione di una versione di valutazione gratuita.
