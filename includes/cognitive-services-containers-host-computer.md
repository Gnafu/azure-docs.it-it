Il **host** è un computer basato su x64 che esegue il contenitore docker. Può essere un computer nell'ambiente locale o un servizio di hosting Docker in Azure, tra cui:

* [Servizio Azure Kubernetes](../articles/aks/index.yml)
* [Istanze di Azure Container](../articles/container-instances/index.yml)
* Cluster [Kubernetes](https://kubernetes.io/) distribuito in [Azure Stack](../articles/azure-stack/index.yml). Per altre informazioni, vedere [Deploy Kubernetes to Azure Stack](../articles/azure-stack/user/azure-stack-solution-template-kubernetes-deploy.md) (Distribuire Kubernetes in Azure Stack).
