È necessario avere un [account delle API Servizi cognitivi](../articles/cognitive-services/cognitive-services-apis-create-account.md) con accesso all'API Riconoscimento input penna. Se non si ha una sottoscrizione di Azure, è possibile [creare un account](https://azure.microsoft.com/try/cognitive-services/) gratuito. È possibile ottenere la chiave di sottoscrizione dal [portale di Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md#get-the-keys-for-your-subscription) dopo la creazione dell'account oppure il [sito Web Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) dopo l'attivazione di una versione di valutazione gratuita.
