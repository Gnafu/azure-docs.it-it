Se l'utente non ha familiarità con i Servizi cognitivi o con Azure, creare una [chiave temporanea](https://azure.microsoft.com/try/cognitive-services/). Se l'utente ha un account Azure, creare una nuova risorsa nel portale. Le query dell'endpoint Language Understanding supportano due tipi di risorse di Azure: **Language Understanding** o la risorsa **Servizio cognitivo**. È necessario quando si [assegna](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md) la chiave all'app.
