> [!IMPORTANT]
> Questo articolo riguarda contenuti che richiedono l'interfaccia della riga di comando di Azure classica. Questa è una versione meno recente dell'interfaccia della riga di comando di Azure, da usare solo con il modello di distribuzione classico di Azure.
>
> L'interfaccia della riga di comando classica può essere installata insieme alla versione moderna di Azure, ma è consigliabile l'uso dell'interfaccia della riga di comando di Azure per tutti gli script e le distribuzioni nuove.
> Per installare l'interfaccia della riga di comando classica, vedere [Installare l'interfaccia della riga di comando di Azure classica](/cli/azure/install-classic-cli) e, per installare l'interfaccia della riga di comando attuale, vedere [installare l'interfaccia della riga di comando di Azure](/cli/azure/install-azure-cli).
