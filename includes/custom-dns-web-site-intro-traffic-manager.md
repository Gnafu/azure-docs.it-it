Quando si usa Gestione traffico di Microsoft Azure per bilanciare il carico del traffico diretto al sito Web di Azure, tale sito Web è accessibile tramite il nome di dominio **\*.trafficmanager.net** assegnato da Azure. È inoltre possibile associare un nome di dominio personalizzato, ad esempio www\.contoso.com, con il sito Web per fornire un nome di dominio più facilmente riconoscibile per gli utenti.
