- Per accedere al servizio dati al bordo casella, l'organizzazione deve avere una sottoscrizione Enterprise Agreement (EA) o Cloud Solution Provider (CSP). Per altre informazioni, vedere [iscriversi a una sottoscrizione di Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Poiché questo servizio di gestione è ospitato in Azure, viene protetto dalle funzionalità di sicurezza di Azure. Per altre informazioni sulle funzionalità di sicurezza fornite da Azure, visitare il [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Per le operazioni di gestione di SDK, è possibile ottenere la chiave di crittografia per la ricorsa nel **le proprietà del dispositivo**. È possibile visualizzare la chiave di crittografia solo se si dispone delle autorizzazioni per l'API Graph di risorse.
