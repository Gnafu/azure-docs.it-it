> [!IMPORTANT]
> Questa funzionalità è in anteprima. Per usarla, è necessario installare l'[estensione Griglia di eventi](/cli/azure/azure-cli-extensions-list) per l'[interfaccia della riga di comando di Azure](/cli/azure/install-azure-cli). È possibile installarla con `az extension add --name eventgrid`. Se si è già installata l'estensione, aggiornarla con il comando `az extension update -n eventgrid`. Se si usa l'API REST, assicurarsi di usare `api-version=2018-09-15-preview`.
