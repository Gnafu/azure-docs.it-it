Per attivare i log in streaming per app per le funzioni in Azure:

1. Premere il tasto F1 per aprire il riquadro comandi, quindi cercare ed eseguire il comando **funzioni di Azure: Avviare lo Streaming dei log**.

1. Selezionare l'app per le funzioni in Azure, quindi selezionare **SÃ¬** per abilitare la registrazione dell'applicazione per app per le funzioni.

1. Attivare le funzioni in Azure e notare che i dati di log viene visualizzati nella finestra di Output in Visual Studio Code.

1. Al termine, ricordarsi di eseguire il comando **funzioni di Azure: Arrestare lo Streaming dei log** per disabilitare la registrazione per l'app per le funzioni.
