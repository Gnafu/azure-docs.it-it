## <a name="sign-in-to-azure"></a>Accedere ad Azure

Usare l'interfaccia della riga di comando di Azure per creare le risorse necessarie per ospitare l'app in Azure. Se è possibile eseguire comandi dell'interfaccia della riga di comando di Azure in Cloud Shell, si è già connessi. Per eseguire comandi dell'interfaccia della riga di comando di Azure in locale, accedere alla sottoscrizione di Azure con il comando [az login](/cli/azure/) e seguire le istruzioni visualizzate.

```azurecli
az login
```
