> [!IMPORTANT]
> Questo articolo riguarda contenuti che richiedono l'interfaccia della riga di comando classica di Azure. La versione corrente dell'interfaccia della riga di comando di Azure non supporta le funzionalità descritte in questo articolo. È quindi necessario usare l'interfaccia della riga di comando classica. 
> 
> L'interfaccia della riga di comando classica può essere installata insieme alla versione moderna dell'interfaccia della riga di comando di Azure, ma è consigliabile l'uso dell'interfaccia della riga di comando di Azure per tutti i nuovi script e le nuove distribuzioni in cui sia disponibile il supporto della funzionalità. Per installare l'interfaccia della riga di comando classica, vedere [Installare l'interfaccia della riga di comando classica di Azure](https://docs.microsoft.com/cli/azure/install-classic-cli?view=azure-cli-latest). 
> Per installare l'interfaccia della riga di comando corrente, vedere [Installare l'interfaccia della riga di comando di Azure](https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest).
