## <a name="create-a-cloud-spatial-anchor"></a>Creare un punto di ancoraggio cloud spaziali

Per creare un punto di ancoraggio cloud, prima di tutto creare un ancoraggio nel sistema di AR della piattaforma e quindi creare una controparte di cloud. Si utilizza il `CreateAnchorAsync()` (metodo).
