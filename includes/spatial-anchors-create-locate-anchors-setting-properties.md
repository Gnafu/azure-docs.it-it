## <a name="set-properties"></a>Impostare le proprietà

È possibile scegliere di aggiungere alcune proprietà quando si salvano i punti di ancoraggio cloud. Ad esempio il tipo di oggetto in fase di proprietà salvate o di base, ad esempio se è necessario abilitarla per l'interazione. In questo modo può essere utile all'individuazione: è possibile immediatamente il rendering dell'oggetto per l'utente, ad esempio un frame di immagini con contenuto vuoto. Quindi, un download in background Ottiene i dettagli di stato aggiuntivi, ad esempio, l'immagine da visualizzare nel frame.
