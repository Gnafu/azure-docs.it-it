> [!IMPORTANT]
> Questo articolo riguarda la versione ospitata dal cliente dei *processi di database elastico*. I processi di database elastico verranno deprecati e sostituiti con nuovi **processi di database elastico** ospitati in Azure. Per i nuovi processi, usare [la versione più recente dei **processi di database elastico**](../articles/sql-database/elastic-jobs-overview.md). Se attualmente si usano i precedenti processi ospitati dal cliente, vedere [Eseguire la migrazione ai nuovi processi di database elastico](../articles/sql-database/elastic-jobs-migrate.md) per ottenere indicazioni e script di migrazione per un rapido aggiornamento alla versione più recente.
