> [!NOTE]
>
> Questo articolo è stato aggiornato di recente per usare i comandi del nuovo modulo Az di Azure PowerShell. Si tratta del nuovo modulo di PowerShell per Azure, compatibile con le versioni 5.x e 6 di PowerShell. AzureRM continuerà a ricevere gli aggiornamenti critici per la correzione di bug, ma le nuove funzionalità verranno inserite solo nel modulo Az.
>
> * Per altre informazioni sul modulo Az e sul supporto per il modulo AzureRM, vedere [Introduzione del nuovo modulo Az di Azure PowerShell](/powershell/azure/new-azureps-module-az).
> * Il modulo Az è già disponibile in Cloud Shell, con la compatibilità con AzureRM abilitata automaticamente.
>   Per istruzioni sull'installazione del modulo Az, vedere [Installare Azure PowerShell](/powershell/azure/install-az-ps).
> * Per abilitare i comandi di compatibilità di AzureRM, eseguire `Enable-AzureRmAlias` dopo l'installazione.
>   Per altre informazioni sulla compatibilità, vedere [Eseguire la migrazione da AzureRM ad Az di Azure PowerShell](/powershell/azure/migrate-from-azurerm-to-az).
