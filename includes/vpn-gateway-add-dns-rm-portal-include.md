1. Nella sezione **Impostazioni** della pagina della rete virtuale selezionare **Server DNS** per aprire la **pagina corrispondente**.

2. Nella pagina **Server DNS** inserire i valori seguenti:

   - **Server DNS**: selezionare **Personalizzato**.

   - **Aggiungi server DNS:** immettere l'indirizzo IP del server DNS che si vuole usare per la risoluzione dei nomi.

3. Dopo aver aggiunto i server DNS, selezionare **Salva**.

    ![Specificare un server DNS](./media/vpn-gateway-add-dns-rm-portal-include/specify-dns-server.png "Specificare un server DNS")
