1. Nella pagina **Impostazioni** della rete virtuale passare a **Server DNS** e fare clic per aprire la **pagina corrispondente**.

    ![Specificare un server DNS](./media/vpn-gateway-specify-dns-portal-include/specify-dns-server.png "Specificare un server DNS")

  - **Server DNS:** selezionare **Personalizzato**.
  - **Aggiungi server DNS:** immettere l'indirizzo IP del server DNS che si vuole usare per la risoluzione dei nomi.

2. Dopo aver aggiunto i server DNS, fare clic su **Salva** nella parte superiore della pagina.
